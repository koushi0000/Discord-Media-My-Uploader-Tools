# Discord Media My Uploader Tools
対応プラットフォーム
PCOnly/Windows10,11のみ
Python

# 説明
DiscordのMedia制限を外部的に回避して
動画や写真を貼れるようにしたツールです

# 使い方
ダウンロード後解凍してお好きなフォルダーに入れてください。
exeを起動してファイルをアップロードすれば自動的にリンクがPCのクリップボードへコピーされます。アップロードしたファイルは履歴で見れるようになっています
ファイル名選択、ダブルクリックでリンクコピー可能です。
